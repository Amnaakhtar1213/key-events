##   KEYS-EVENTS

This project shows us how to move an HTML element around the screen using keyboard arrow keys.It also change box's emoji and background colour depending on which key is press down or released.In this project i came to know that arrow-left and arrow-right is not a real DOM events. To keep element on the screen i dont use preventDefault() because the element disappear when we press one key for long time.

-------

##   Features

1. ArrowUp: decreases y (moves box upward).

2. ArrowDown: increases y (moves box downward).

3. ArrowLeft: decreases x (moves box left).

4. ArrowRight: increases x (moves box right).

5. Emoji and colour change on key press/release.

6. Smooth movement is four direction.

7. Easy to customize moveAmount for faster/slower    movement.
